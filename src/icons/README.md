# Extension Icons

This directory should contain the extension icons in PNG format:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Icon Requirements

- **Format**: PNG
- **Background**: Transparent or solid color
- **Style**: Simple, recognizable design
- **Colors**: Should work on both light and dark backgrounds

## Suggested Design

Consider an icon that represents:
- Email/outreach (envelope, @ symbol)
- Speed/efficiency (lightning bolt, arrow)
- AI/automation (robot, brain, sparkles)
- Professional networking (handshake, connection nodes)

## Tools for Icon Creation

- [Figma](https://figma.com) - Free design tool
- [Canva](https://canva.com) - Icon templates
- [Iconify](https://iconify.design/) - Free icon library
- [GIMP](https://gimp.org) - Free image editor

## Temporary Solution

For development/testing, you can use any PNG files with the correct names and sizes. The extension will work without custom icons (Chrome will show a default puzzle piece icon).
